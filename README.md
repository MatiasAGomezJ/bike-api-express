1. Instalar las dependencias.

```bash
$ npm install
```

2. Configurar el archivo `.env`.

3. Levantar el proyecto.

```bash
$ npm run serverstart
```

-   Front, todas son accesibles desde el indice:
    -   `localhost:3000`
    -   `/bike`
